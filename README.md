<!-- GETTING STARTED -->

### Installation

## Using qpm-rust

1. Run in your mod directory through Powershell
```sh
qpm-rust dependency add QuestModDebugTools
```
2. Then run 
```sh
qpm-rust restore
```

## Without qpm-rust

1. In your mod directory, run
```sh
cd extern
```
2. Then run
```sh
git clone
